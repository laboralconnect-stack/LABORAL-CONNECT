# Laboral Connect

Web estática lista para desplegar en Vercel.

## Antes de publicar

1. Completar los campos `[completar]` de las páginas legales.
2. Cuando dispongas de número profesional, convierte el bloque de WhatsApp de `index.html` en un enlace con formato `https://wa.me/34NUMERO`.
3. Sustituir `[completar]` en `GA_MEASUREMENT_ID`, dentro de `assets/main.js`, por el ID real de GA4.
4. Revisar los textos legales con un profesional y formalizar el alta necesaria antes de captar clientes.

El correo `laboralconnect@gmail.com` y el Instagram `@laboral_connect` ya están configurados.
